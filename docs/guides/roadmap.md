# Roadmap

## Phase 1 — ISO/IEC 27001 Automation Core

Scope:
- Full Annex A (2022) control catalog (93 controls)
- Risk lifecycle with deterministic scoring
- Evidence lifecycle with expiry/review cadence
- Compliance scoring engine
- Continuous daily monitoring job
- Production-safe scheduler (cron-driven)
- Statement of Applicability (SoA) PDF export
- Tenant-isolated API enforcement
- Release-gated CI validation

## Phase 2 — Multi-Framework Expansion
- SOC 2 automation pack
- PCI and HIPAA framework packs
- Cross-framework control mapping and reporting

## Phase 3 — Continuous Compliance Intelligence
- Continuous controls monitoring expansion
- Auditor collaboration workflows
- Advanced risk modeling

## Phase 4 — Regulatory Ingestion Engine
- Regulatory source ingestion pipeline
- Structured obligation extraction and change tracking
- Region- and industry-specific update feeds

## Phase 5 — AI Automation + Co-Pilot
- AI-guided remediation orchestration
- Evidence quality co-pilot and recommendations
- Workflow automation assistant across compliance operations
